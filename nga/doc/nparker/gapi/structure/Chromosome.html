Interface Chromosome

public interface Chromosome
Class Chromosome unifies chromosomes across genetic algorithms by giving a uniform access to all chromsomes in a particular population. Generically, we want to be able to represent a Chromosome in a genetic algorithm population as an array of any type of genes, not just bits. This interface allows us to represent a chromsome as a disparate array of any objects unified within the Chromosome structure.

getGenePattern

public Object[] getGenePattern()
Returns the entire gene pattern of this Chromosome. The entire genetic makeup of the chromosome is treated as an array of Objects for review purposes by outsiders. Obviously, a problem-specific chromosome implementation may be made up as it likes, but the implementation must be able to translate to an array.
Returns:

getGenePattern

public Object[] getGenePattern(int startIndex,
                               int endIndex)
Returns the gene pattern of this Chromosome from startIndex to endIndex, inclusive. Whether or not specific implementations of this interface return an array from startIndex to endIndex, inclusive, is not a hard and fast rule as the main algorithm in the class MainTest never makes use of this method. The range can be modified to be from startIndex to endIndex - 1, inclusive, but implementations that deviate from the original specification should make note of such a deviation.
Parameters:
startIndex - the index of the first gene to be returned.
endIndex - the index of the last gene to be returned.
Returns:
the gene pattern of this Chromosome from startIndex to endIndex, inclusive.

length

public int length()
Returns the number of genes in this Chromosome. This is intended to be interpretted as the length of the Object array that is returned by the getGenePattern(). That is, the value returned by this method should be consistent with the length member of that particular array. Deviations or alterations from this interpretation of the implementation of this method in problem-specific implementations of this interface should make note of their alterations to this definition.
Returns:
the number of genes in this Chromosome.
