Steiner8DChromosome

public Steiner8DChromosome()
Creates a new bit-wise chromosome which is tailored to solve the 8-point Steiner problem. The bit-wise chromosome has an internal length of 88 bits. This uses Java's random number generator in the java.lang.Math to determine the initial settings of each bit in the bit string and not the random number generator set in the GeneticApplication class in this package.
