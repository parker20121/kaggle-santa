Class SteinerFitnessFunction

public class SteinerFitnessFunction
implements FitnessFunction

The fitness function for SteinerPointChromosomes where the distances between all of the points on the unit cube and the steiner points are summed. The total distance between all of these points represents the fitness for this chromosome.

This fitness function only works on chromosomes of class SteinerPointChromosome. Any other class will not get results using this fitness function since this function relies upon the particular gene pattern of the chromosome to be of a certain composition.

Method Detail

evaluateFitness

public double evaluateFitness(Chromosome chromosome)
Returns the total distance between the steiner points and the points on the unit square that those points are connected to. Thus, the total distance is a measure of the fitness of the given chromosome. If the given chromosome does not have the correct connection array structure where every point is connected to a steiner point then the total distance returned will be assigned Double.POSITIVE_INFINITY. The chromsome must be a SteinerPointChromsome in order for it to be evaluated correctly.
Parameters:
chromosome - a SteinerPointChromosome to be evaluated.
Returns:
the total distance between the steiner points and the points on the unit square as a measure of the fitness, or Double.POSITIVE_INFINITY if the chromosome's connection structure is in error.
