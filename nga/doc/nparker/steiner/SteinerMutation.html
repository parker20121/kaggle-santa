Class SteinerMutation

public class SteinerMutation
implements MutationStrategy
Mutation strategy for the steiner point problem. Considering the individual genes in a SteinerPointChromosome we can introduce variation into the particular gene by either flipping the value if the gene is a Boolean or varying it across the given range if it is a Float.
